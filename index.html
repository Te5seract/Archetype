File Structure & Purpose

  • src/main.js is the JS entry point. The Workspace class is instantiated there and passed a bunch of other classes that relate to pages
  • src/classes/Workspace.js This class is instantiating classses and then running their methods based on method types and routing.
  • src/pages/* These are classes that correspond to HTML pages. In the files here there will be a reference to my personal library Xenium: x("@x-toggled").events("click", function () {});
